# MODELO-PROSPERIDAD-SOSTENIBLE
El Modelo de Prosperidad Sostenible (MPS) es una herramienta macroeconómica que mide la eficiencia institucional frente al gasto público. Armoniza datos del BM, FMI, BCE y OCDE para calcular el parasitismo (TPL) y la prosperidad neta (PN) real, auditando el coste de la estructura estatal y la redistribución frente a la libertad económica.
